what_i_learned: |
  This week, I learned about function inputs, parameters, and arguments in Python. A parameter is a variable that is placed inside the parentheses when defining a function, while an argument is the value passed into the function when it is called. Understanding the difference between parameters and arguments helped me better understand how functions receive and process information. I also learned how functions can return values and be reused multiple times with different inputs. The lottery number example showed me how parameters can make functions more flexible and useful. These concepts helped strengthen my understanding of writing organized and efficient code. I also learned that many tools used with strings can be applied to lists. The len() function can be used to find the number of items in a list, and the in keyword can be used to check whether a specific item exists in a list. In addition, I learned how to upload a dataset from a link into Google Colab and unzip the file using Python. This was an important skill because many data science and research projects require downloading and preparing datasets before analysis can begin. Learning how to work with datasets in Google Colab gave me more confidence in using coding tools for research and data management. Overall, I gained valuable knowledge about Python functions, lists, and handling datasets in a cloud-based coding environment.
