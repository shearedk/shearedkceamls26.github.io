what_i_learned: |
  Today I learned how computers make decisions by following step-by-step instructions called algorithms. I understand that algorithms are clear sets of instructions that a computer follows to complete a task. We practiced breaking down problems like a wage calculator into structured steps that are easy to translate into code. I also learned that pseudocode is used to plan programs before writing actual Python code. Pseudocode helps organize thoughts in a simple way without worrying about syntax. This makes it easier to design programs that are accurate and logical before coding. I also learned how conditional statements help computers make decisions based on different inputs. The “if” statement allows a program to run certain code only when a condition is true. I learned that indentation is very important in Python because it shows what code belongs inside the condition. Without proper indentation, the program would not run correctly. I also saw how Boolean values like True and False are used to control program behavior. Overall, I learned how algorithms, pseudocode, and conditionals work together in programming.
