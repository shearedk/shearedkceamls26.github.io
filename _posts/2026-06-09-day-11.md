what_i_learned: |
  Today, I learned about the differences between 1D Convolutional Neural Networks (CNNs) and 2D Convolutional Neural Networks (CNNs). A 1D CNN is commonly used for data that exists in a sequence, such as ECG signals, audio recordings, and other time-series data. It moves across the data in one direction and learns important patterns over time. In contrast, a 2D CNN is designed for image data and moves both horizontally and vertically across an image. This allows the model to detect features such as edges, shapes, textures, and objects. I also learned that both types of CNNs use filters to identify important features and then use those features to make predictions. Understanding these concepts helped me see how deep learning models can be applied to different types of data depending on the problem being solved. During the literature review session with Dr. Anita Pandey, I learned the importance of using transition phrases to improve the flow and organization of academic writing. Some examples of transition phrases include "for example," "in addition," "however," "therefore," and "as a result." These phrases help connect ideas and make writing easier for readers to follow. For instance, "however" can be used to introduce a contrasting idea, while "therefore" can be used to show a conclusion based on previous information. I also watched a YouTube video about Object-Oriented Programming (OOP) in Python. From the video, I learned that OOP is a programming approach that organizes code into classes and objects. For example, a class called "Student" can contain attributes such as a student's name and age, while objects are created from that class to represent individual students.

Although these topics were challenging, reviewing examples and practicing helped improve my understanding.
